# Model: Audit Log

**Purpose:** Security audit trail

**Storage:** Database `audit_log` table

---

## Zod Schema

```typescript
import { z } from "zod";

export const AuditLogSchema = z.object({
  id: z.string().uuid(),
  spaceId: z.string().uuid(),
  shareId: z.string().uuid().optional(),
  userId: z.string().uuid().optional(),
  action: z.string(),
  path: z.string().optional(),
  details: z.record(z.any()).optional(),
  createdAt: z.string().datetime(),
});

export type AuditLog = z.infer<typeof AuditLogSchema>;
```

---

## Fields

### `id`

- **Type:** `UUID`
- **Description:** Unique identifier for this audit entry
- **Generated by:** Spaces Service database

### `spaceId`

- **Type:** `UUID`
- **Description:** Space this action relates to
- **References:** `spaces.id`

### `shareId`

- **Type:** `UUID | undefined`
- **Description:** Share link used (if applicable)
- **References:** `shares.id`

### `userId`

- **Type:** `UUID | undefined`
- **Description:** User who performed action (if applicable)
- **References:** `users.id`
- **Note:** `undefined` for anonymous access

### `action`

- **Type:** `string`
- **Description:** Action performed
- **Examples:** `"file.read"`, `"file.write"`, `"chat.send"`, `"share.create"`

### `path`

- **Type:** `string | undefined`
- **Description:** File path involved (for file operations)
- **Example:** `"Vacations/Maine.md"`

### `details`

- **Type:** `JSONB | undefined`
- **Description:** Additional details about the action
- **Example:** `{ "bytes": 1024, "lineCount": 42 }`

### `createdAt`

- **Type:** `Date` (ISO 8601)
- **Description:** When this action occurred

## Logged Actions

| Action | Description | Details |
|--------|-------------|---------|
| `space.create` | Space registered | `{ agentId, path }` |
| `space.delete` | Space unregistered | `{ reason }` |
| `share.create` | Share link created | `{ role, expiresAt }` |
| `share.revoke` | Share link revoked | `{ reason }` |
| `share.access` | Share link accessed | `{ ipAddress }` |
| `file.read` | File viewed | `{ bytes }` |
| `file.write` | File modified | `{ bytes, lines }` |
| `chat.send` | Chat message sent | `{ messageLength }` |
| `session.start` | Session started | `{ agentSessionId }` |
| `session.end` | Session ended | `{ duration }` |

## Database Schema

```sql
CREATE TABLE audit_log (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    space_id UUID REFERENCES spaces(id) ON DELETE SET NULL,
    share_id UUID REFERENCES shares(id) ON DELETE SET NULL,
    user_id UUID REFERENCES users(id) ON DELETE SET NULL,
    action TEXT NOT NULL,
    path TEXT,
    details JSONB,
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_audit_space ON audit_log(space_id);
CREATE INDEX idx_audit_share ON audit_log(share_id);
CREATE INDEX idx_audit_user ON audit_log(user_id);
CREATE INDEX idx_audit_action ON audit_log(action);
CREATE INDEX idx_audit_time ON audit_log(created_at);
```
