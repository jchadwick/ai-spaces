## Priorities

**MVP (v0.1):**
- Space discovery
- Share link creation/validation
- Token-based authentication
- Read-only file browser
- Basic scoped chat

**Post-MVP:**
- File editing
- Multi-user collaboration
- Owner management UI
- Chat history persistence
